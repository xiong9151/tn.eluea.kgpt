## Features

- **Superior Usability**: Modern UI using **MD3 Expressive** design standards
- **Internal Update System**: Auto-updates directly within the app
- **Redesigned Web Search**: 10+ search engines including **Perplexity AI**
- **App Trigger**: Switch between apps using custom trigger commands
- **Advanced Command System**: Execute complex operations with simple commands
- **Latest AI Model Support**: Including **Gemini 1.5/2.0**
- **AI Text Actions**: Select text for Refresh, Improve, Translate, Summarize, Explain, etc.
- **Full Theme Support**: Amoled, Dark, Light modes with winter animation
- **Material You**: Extract colors from wallpaper or set custom seed color
- **Blur Effects**: Adjustable transparency, color, and behavior
- **Full Backup & Restore**: Backup settings, keys, themes, commands, triggers
