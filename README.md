## <img src="https://fonts.gstatic.com/s/e/notoemoji/latest/1f425/512.gif" alt="üê£" width="35" height="35"> Introduction

A special thanks to **Mino260806** for establishing the original idea and foundation upon which this project was developed. Without their initial work, this repository would not exist today.

**KGPT** is a powerful tool designed to integrate your favorite AI models directly into your keyboard, allowing you to use them seamlessly across various tasks and activities on your device.

> [!IMPORTANT]
> **This is NOT just a standard update of the original project.**
> This is a complete **structural rebuild and redesign from scratch**. The code has been rewritten with an entirely different infrastructure, changing the core logic and introducing features that never existed before.

## <img src="https://fonts.gstatic.com/s/e/notoemoji/latest/1f423/512.gif" alt="aaa" width="35" height="35"> Features

This rebuild includes a vast array of new capabilities (too many to list everything, but here are the highlights):

- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Objects/Luggage.webp" alt="Luggage" width="25" height="25" /> **Superior Usability**: A fully developed application with a modern, advanced user interface using the latest **MD3 Expressive** design standards.
- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Activity/Artist%20Palette.webp" alt="Artist Palette" width="25" height="25" /> **Elegant & Modern Design**: Smart design concepts that make using the app a visually and practically enjoyable experience.
- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Animals%20and%20Nature/Cloud.webp" alt="Cloud" width="25" height="25" /> **Internal Update System**: Automatically keeps you informed and updated with the latest versions directly within the app.
- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Objects/Open%20Book.webp" alt="Open Book" width="25" height="25" /> **Detailed Changelog**: Built-in system to display detailed changelogs for every update.
- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Objects/Magnifying%20Glass%20Tilted%20Right.webp" alt="Magnifying Glass Tilted Right" width="25" height="25" /> **Redesigned Web Search**: Support for over 10 search engines (including the absolute best) and AI-powered search tools like **Perplexity AI**.
- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Travel%20and%20Places/Rocket.webp" alt="Rocket" width="25" height="25" /> **App Trigger**: Instantly switch between apps using a custom trigger command you define.
- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Objects/Toolbox.webp" alt="Toolbox" width="25" height="25" /> **Unrestricted Customization**: "Open limits" philosophy allowing you to customize triggers exactly how you want without restrictions.
- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Objects/Keyboard.webp" alt="Keyboard" width="25" height="25" /> **Advanced Command System**: Execute complex operations that would usually require extensive typing with a simple, single command while typing.
- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Objects/Mobile%20Phone.webp" alt="Mobile Phone" width="25" height="25" /> **Evolution of Floating Windows**: Redesigned legacy floating windows (location, appearance, interaction) to help users from the original version adapt to the new experience.
- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Objects/Gem%20Stone.webp" alt="Gem Stone" width="25" height="25" /> **Latest AI Model Support**: Updated support for all AI types, including the newest models like **Gemini 1.5/2.0**.
- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Objects/Memo.webp" alt="Memo" width="25" height="25" /> **AI Text Actions**: Select any text to access default options (Refresh, Improve, Translate, Summarize, Explain, Formal, Casual, etc.) and add your own advanced custom actions.
- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Activity/Mirror%20Ball.webp" alt="Mirror Ball" width="25" height="25" /> **Full Theme Support**: Complete support for **Amoled**, **Dark**, and **Light** modes, featuring a special winter animation.
- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Activity/Magic%20Wand.webp" alt="Magic Wand" width="25" height="25" /> **Material You Customization**: Extract colors directly from your wallpaper (on supported Android versions) or set a **custom seed color**.
- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Activity/Jack%20O%20Lantern.webp" alt="Jack O Lantern" width="25" height="25" /> **Blur Effects**: Adjustable blur transparency, color, and behavior (derived from background or custom seed).
- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Symbols/Up%20Button.webp" alt="Up Button" width="25" height="25" /> **Update Settings**: Control download paths and perform manual update checks.
- <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Objects/Briefcase.webp" alt="Briefcase" width="25" height="25" /> **Full Backup & Restore**: Backup literally everything‚Äîsettings, keys, themes, commands, triggers‚Äîand choose exactly what to restore.
